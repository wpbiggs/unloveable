# Intake

This folder contains prompts for a Foundry-style intake:

1) Ask exactly 5 clarifying questions.
2) Generate canonical docs (`spec.md`, `brand-and-ux-spec.md`, `architecture.md`, `implementation-plan.md`, `prompt.md`).

Use `templates/unloveable/intake-loop.sh` (or the repo root `intake-loop.sh`) to run this flow.
